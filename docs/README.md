# VySol Docs Folder

This folder contains project documentation assets and documentation-related files.

Retype-specific site files live in `docs/documentation`. General docs folder files, such as this README, the QUICKSTART, and the changelog, stay directly inside `docs`.

## Structure

- `documentation/` contains the Retype documentation site.
- `assets/` contains documentation images and media.
- `QUICKSTART.md` contains repository setup and launch steps.
- `CHANGELOG.md` tracks notable project changes.
