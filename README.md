# VySol

VySol is an early-stage roleplay and world-simulation app. The project starts with tools for building and exploring fictional worlds, with a longer-term direction toward digital twin simulation research.

## Documentation

Project documentation lives in `docs/documentation` and is published with Retype through GitHub Pages.
